BleedBlue React is a React based UI component library made for **component driven
development (CDD)** of, responsive and attractive web applications in minutes.

The component library has 2 levels of UI components.

- **Fragments** : The smallest independent unit of UI
- **Components** : Comibnation of fragments to make a complex component
